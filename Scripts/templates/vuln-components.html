This script will gather all JavaScript files loaded by a page and scan them for known vulnerabilities. This is done with the help of RetireJS (Python). Additionally, it will perform banner grabbing to determine the web server used, for example nginx or Apache, and potentially its version and searches the National Vulnerability Database (NVD) for related CVEs.

About:

Using Components with Known Vulnerabilities is a very widespread issue among web applications. Many development teams may not even understand what components or modules they are using due to a large amount of components being implemented. Keeping such components up-to-date is an even harder task.

Impact:

Some vulnerabilities lead to only minor impacts, however, some of the largest breaches that have occured leveraged known vulnerabilities in components. The impact depends on the assets at risk.
