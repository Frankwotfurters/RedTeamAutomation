Robotic Process Automation (RPA) Demonstration is available during the scan. The RPA tool will take a screenshot of the vulnerable admin page and save it to the user's system.

About:

Admin Interface Scanner is a tool that scans through a website to locate admin interfaces with common directory names. The scanner will loop through a curated wordlist of commonly compromised admin names, and test connections towards the website. If a connection is opened with a name from the list, the script will detect that and show the results at the end of the scan.

Impact:

This vulnerability is under the OWASP Top 10 - Security Misconfigurations. Adversaries can potentially gain unauthorized access to the admin panel and alter configurations to negatively impact the website.
